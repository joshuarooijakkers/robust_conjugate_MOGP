# robust_conjugate_MOGP

This repository contains the code and experiments for the paper, *Robust and Conjugate Multi-Output Gaussian Processes*. It provides several robust multi-output Gaussian process regression methods.

[ðŸ“„ Read the paper on arXiv](https://arxiv.org/abs/2510.26401)

This repository includes preprocessing scripts, model training code, and evaluation utilities used to reproduce the main results.

## Quick Start

```bash
git clone https://github.com/joshuarooijakkers/robust_conjugate_MOGP.git
cd robust_conjugate_MOGP
```
